# CoT Overthinking

This repo contains baseline CoT responses from Qwen3-32B to a subset of the MMLU-Pro dataset, with and without giving the model the answer options. I investigate how the model's answer changes over the course of the CoT and in particular what causes it from having the correct answer to the incorrect answer.
